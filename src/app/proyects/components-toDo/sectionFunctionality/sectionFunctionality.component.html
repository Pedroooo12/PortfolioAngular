Funcionalidades.

Incluye diversas funcionalidades para mejorar la organización y gestión de tareas. Este sistema permite a los usuarios interactuar de manera eficiente con sus listas de tareas.

Navegación Intuitiva:

Implementación de flechas hacia arriba y hacia abajo en cada columna para clasificar las tareas según su nivel de importancia, facilitando la priorización.

Arrastrar y Soltar:

Funcionalidad de arrastrar y soltar que permite mover tareas entre columnas. Los cambios se guardan automáticamente y se mantienen incluso al refrescar la página. Esta funcionalidad la he incluido mediante Angular Material.

Filtrado Avanzado:

Opción para filtrar tareas por categoría, nivel de importancia o una combinación de ambos, permitiendo a los usuarios encontrar rápidamente lo que necesitan.

Finalización de Tareas:

  • Opción para marcar tareas como "Hechas". Los usuarios pueden finalizar tareas individualmente.

  • Utilizar un icono para completar todas las tareas de una vez.

Buscador Dinámico:

Un buscador que permite a los usuarios escribir cualquier letra, mostrando un listado de tareas que coinciden en tiempo real. La lista se actualiza automáticamente mientras se escribe, mejorando la accesibilidad y la rapidez en la búsqueda.
