Inicio de sesión:

En esta sección podrás iniciar sesión en el sistema, para poder acceder a las funcionalidades dependiendo de si eres usuario o administrador, te mandará a un panel u otro:

  • Tienes que tener el usuario y la contraseña correctos para poder entrar.

  • Si tienes algún error te notificará en pantalla, dependiendo de si no existe el usuario o la contraseña es incorrecta.

  • El inicio de sesión va por JWT por lo que se le manda al usuario un token exclusivo para entrar para él y para esa sesión en concreto.
