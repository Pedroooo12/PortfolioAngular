Crear empleado.

Para que el empleado se cree correctamente tenemos que realizar estos 3 pasos, podemos saltar de un paso a otro sin necesidad de rellenar los datos:

  1. Crear el empleado con su nombre, email y contraseña, este paso es totalmente obligatorio.

  2. Crear la información personal del empleado como DNI, calle... es una opción opcional por lo que puedes crearlo o no.

  3. La última es parte es una confirmación de que todo está correcto mostrando todo lo que hemos creado y si queremos modificar o eliminar alguna de las dos partes.
