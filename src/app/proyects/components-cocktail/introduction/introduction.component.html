Introducción.

Este proyecto es una aplicación web con la última version de Angular y sus nuevas funcionalidades como señales, la cual tiene información sobre cócteles que son recogidos de una API, incluye un buscador de cócteles y botones para mostrar los cócteles dependiendo de sus ingredientes, alcoholes… Además tiene la opción de poder descargar la aplicación en tu móvil u ordenador.
